The L root name server is run by ICANN and until recently, this server had an address of 198.32.64.12. On 24 October 2007, ICANN announced it was changing to the address to 199.7.83.42, effective 1 November 2007.

…if you inadvertently went looking for the old L root name server [after] this time, you might have ended up at any one of four very different places!
